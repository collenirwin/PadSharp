![PadSharp Example Image](https://github.com/collenirwin/PadSharp/raw/master/images/pad#_summary.png)

## Notable Features
- 2 themes (Light & Dark)
- Spelling check
- Definition lookup
- A few basic markdown-esque style rules
- Checklists
- Sort, reverse lines
- Find & Replace with Regular Expressions
- Always-on-top toggle
